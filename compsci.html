Java

My bread & butter.

Java was the first programming language I learned and my introduction to Computer Science. The source code depicted below is from a Java project for a Library system and database. My experience at public libraries suggested the system was trivial. However, implementing the code demanded a fundamental understanding of books, the people checking them in/out, and a library database as well as how they interact with each other. This project also encouraged using objects in interesting way, such as creating phone number objects as a means to check a book in/out associated with a user, or generic object "holders." Library book objects, an extention of book objects, included holders and calendar objects as attributes to perform various methods within the class... Honestly I could bore you with the details, but just know this is my favorite Java project!

Python

For Scientists.

Apart of the PRISM (Andvanced Program for Integrated Math and Science) program at the University of Miami, I took a lot of Physcial Science and Mathematics courses along side Comp-Sci. From Organic Chemistry to Biology to Computational Science, I understood the demand for computer porgrams that can efficiently interpret data in a meaningful way. Take for instance, interpreting genomic sequences, where DNA sequences (ATACGGAGTC...) can include MILLIONS of base pairs. Motifs, such as "TATA Boxes," a six-nucleotide sequence part of a promoter site on DNA, allows for transcription to occur. The program shown above uses regular expressions to find all TATA boxes in a DNA sequence and reports their types and locations - parsing through endless DNA strings to find patterns in a breeze.
