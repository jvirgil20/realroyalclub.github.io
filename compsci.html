Java

The first programming language I learned and my introduction to Computer Science. My bread & butter.

The source code depicted is from a Java project for a Library system and database. This project is one of my favorites since it required me to think harder than I imagined I would in my life about how a Library (a collection of books) operates. My expereince at public libraries suggested the system was trivial. However, implementing the code demanded a fundamental understanding of books, the people checking them in/out, and a library database as well as how they interact with each other. This project also encouraged using objects in interesting way, such as creating phone number objects as a means to check a book in/out associated with a user, or generic object "holders." Library book objects, an extention of book objects, included holders and calendar objects as attributes to perform various methods within the class ...Honestly I could bore you with the details, but just know this is my favorite Java project.
